索引引擎 ElasticSearch

## 基本介绍

当我们基于数据库的检索达到瓶颈时，比如数据库中数据量特别大，SQL查询特别慢。或者基于数据库无法实现分词检索时，这时可以考虑使用索引引擎。

## ElasticSearch 的安装

ES 的安装常见的有：Windows 下的安装，Mac 下的安装，Linux 下的安装，以及 Docker 的安装。

## 相关产品

概念 ELK，即 ElasticSearch、Logstash、Kibana 三个产品的缩写。
